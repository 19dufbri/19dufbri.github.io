Projects

The backbone of my work is centered around building projects to learn about new technologies or push my knowledge beyond what it was before. Many of these project, shown below, are focused on building some form of software solution to a challenging problem.

FEH Robot

A fully functional autonomous robot tasked with navigating a course and competing in a competition. Modeled in Solidworks and programmed in C++ as part of the Fundamentals of Engineering Honors Robot Program.

6502 Virtual Machine

Model of the commercially successful MOS 6502 CPU used in many home computing systems from the 70s and 80s. An exploration of emulation and computer internals in a fully custom C project.

Software Design Project - FEHPacman

A regimentation of the classic video game Pacman. Designed to work on the Proteus microcontoller board with a scoreboard and increasing difficulty. Written in C++ as a part of the Fundamentals of Engineering Honors program.
